# 个人网站规划方案

## 技术栈
- **Astro + Tailwind CSS**
- **功能**：Giscus 评论、访问统计、暗色模式
- **部署**：GitHub Pages

---

## 网站结构

```
/
├── src/
│   ├── pages/
│   │   ├── index.astro          # 主页
│   │   ├── experience.astro     # 经历版块（独立页面）
│   │   └── records/
│   │       ├── index.astro      # 个人记录列表
│   │       └── [slug].astro     # 记录详情页
│   ├── layouts/
│   │   └── Layout.astro         # 公共布局
│   ├── content/
│   │   ├── records/             # 个人记录 Markdown
│   │   │   ├── log-1.md
│   │   │   ├── quote-1.md
│   │   │   └── ...
│   │   └── experience/          # 经历 Markdown（独立文件）
│   │       ├── job-current.md
│   │       ├── job-previous.md
│   │       └── ...
│   └── components/
│       ├── Header.astro
│       ├── Footer.astro
│       ├── ThemeToggle.astro
│       └── ExperienceCard.astro
├── public/
│   └── images/                  # 图片资源
│       ├── avatar.jpg
│       ├── exp-1.jpg
│       └── ...
├── astro.config.mjs
└── tailwind.config.mjs
```

## 各版块详细设计

### 1. 主页 (`/index.astro`)

**布局结构：**
```
┌─────────────────────────────────┐
│  导航栏 (Logo + 链接 + 暗色切换)  │
├─────────────────────────────────┤
│                                 │
│  Hero 区域                      │
│  ┌─────────┐  个人简介          │
│  │ 头像    │  姓名、职业         │
│  └─────────┘  一句话介绍         │
│                                 │
│  → 查看我的经历 [可点击跳转]     │
│                                 │
├─────────────────────────────────┤
│  基本信息卡片                    │
│  • 邮箱 / GitHub / 社交媒体      │
│  • 技能标签                      │
│                                 │
├─────────────────────────────────┤
│  最新记录预览 (2-3条)            │
│  ┌──────┐ ┌──────┐ ┌──────┐    │
│  │ 记录1 │ │ 记录2 │ │ 记录3 │    │
│  └──────┘ └──────┘ └──────┘    │
│                                 │
└─────────────────────────────────┘
```

**交互效果：**
- 滚动视差/渐入动画
- "查看我的经历" 按钮 → 跳转到 /experience
- 记录卡片悬停效果

---

### 2. 经历版块 (`/experience.astro`)

**时间轴布局：**
```
┌─────────────────────────────────┐
│  我的工作经历                    │
├─────────────────────────────────┤
│                                 │
│  ● 2023.01 - 至今               │
│    ┌─────────────────────────┐  │
│    │ 公司/职位               │  │
│    │                         │  │
│    │ [插图/图片]             │  │
│    │                         │  │
│    │ 详细描述：               │  │
│    │ • 工作内容1              │  │
│    │ • 工作内容2              │  │
│    │ • 成果/项目...           │  │
│    └─────────────────────────┘  │
│                                 │
│  ● 2021.06 - 2022.12            │
│    ┌─────────────────────────┐  │
│    │ 公司/职位 + 描述...       │  │
│    └─────────────────────────┘  │
│                                 │
└─────────────────────────────────┘
```

**功能：**
- 左侧时间轴，右侧内容卡片
- 支持在 Markdown frontmatter 或配置中定义
- 图片支持：`![alt](/images/exp-1.jpg)`
- 可展开/收起详情

---

### 3. 个人记录 (`/records/`)

**列表页：**
```
┌─────────────────────────────────┐
│  个人记录                        │
├─────────────────────────────────┤
│  [全部] [日志] [文档] [短句]    │
├─────────────────────────────────┤
│  ┌─────────────────────────┐    │
│  │ 2024-01-15  [日志]      │    │
│  │ 标题：今天学习了一新框架  │    │
│  │ 摘要：今天开始学习...     │    │
│  │ [配图缩略图]             │    │
│  └─────────────────────────┘    │
│                                 │
│  ┌─────────────────────────┐    │
│  │ 2024-01-10  [短句]      │    │
│  │ "生活中最美的..."        │    │
│  └─────────────────────────┘    │
│                                 │
└─────────────────────────────────┘
```

**详情页：**
- Markdown 渲染支持
- 图片大图展示（可点击放大）
- 代码高亮
- Giscus 评论区
- 上一篇/下一篇导航

**记录类型（通过 frontmatter 标记）：**
```yaml
---
title: 标题
type: log  # log | doc | quote
date: 2024-01-15
image: /images/xxx.jpg  # 可选
---
```

## Markdown 记录示例

**日志类型：**
```markdown
---
title: 学习 Astro 框架
type: log
date: 2024-01-15
image: /images/astro-post.jpg
---

今天开始学习 Astro，这是一个很棒的静态站点生成器...

![学习笔记截图](/images/notes.png)

继续探索中...
```

**短句类型：**
```markdown
---
title: 关于坚持
type: quote
date: 2024-01-10
---

"不是因为有希望才坚持，而是因为坚持了才有希望。"
```

## 暗色模式实现

- 使用 Tailwind 的 `dark:` 类
- 存储偏好到 localStorage
- 默认跟随系统主题
